RESETPASSWORD PAGE
Used to reset the users password

Header with "Reset password", Image of website logo in top left of screen

-ResetPassword page consists of the following fields
    -Old Password
    -New Password
    -Confirm new Password
    -Submit Password change

    -Forgot password?

If the user knows their current password and just wants to update it then the
old password can be entered followed by the new password in the "New password" and "Confirm Password" fields.
The submit password button can then be clicked to complete the password change which will redirect to the login page.

If the user has forgotten their password they can click the "Forgot password?" button to send an email which will assign them
a temporary password they can use to reset their password.

-From the reset password page you can access the login page
